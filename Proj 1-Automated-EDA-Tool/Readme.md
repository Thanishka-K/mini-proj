# Proj 1-Automated-EDA-Tool
A high-performance Python tool designed to automate the Exploratory Data Analysis (EDA) phase of the ML pipeline. This engine reduces hours of manual plotting and data auditing into a single command, generating comprehensive statistical summaries and visualizations.

# Overview
This tool serves as a "first-look" utility that:
* Identifies missing data and suggests imputation strategies.
* Calculates statistical moments (Skewness, Kurtosis).
* Generates a suite of visualizations (Heatmaps, Histograms, Boxplots).
* Exports insights to a structured directory for immediate analysis.

# Tech Stack
* Language: Python
* Data Manipulation: Pandas, NumPy
* Visualization: Seaborn, Matplotlib
